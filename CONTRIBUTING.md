# Contributing

This repository is **auto-generated** from [mcp.directory](https://mcp.directory). The README, category files, and skills list are produced from a single source of truth and refreshed weekly.

## Adding your MCP server

1. Submit at https://mcp.directory/submit
2. After review (typically <72 hours), your server will appear here on the next refresh.

## Reporting data errors

Open an [issue](https://github.com/devanshug2307/awesome-mcp-servers/issues/new) with:

- The server slug (e.g., `mcp.directory/servers/your-server`)
- What's wrong (description outdated, link broken, miscategorized, etc.)

Corrections are applied upstream at mcp.directory and propagate here on the next regeneration.

## Pull requests

Direct PRs editing the markdown will be closed — changes are overwritten on every refresh. Submit upstream at [mcp.directory/submit](https://mcp.directory/submit) instead.
